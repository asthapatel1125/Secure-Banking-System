**Secure Banking System**

This project is a client-server banking application that simulates real-world ATM operations with a strong focus on security, reliability, and data integrity. The system supports user authentication, deposits, withdrawals, and balance inquiries over encrypted communication channels. It implements AES (Fernet) encryption, HMAC-SHA256 integrity checks, and timestamp-based replay attack prevention to ensure secure transactions. A multi-threaded server handles concurrent clients while maintaining thread-safe audit logs for all activities. The application features a Tkinter-based GUI for user interaction and uses TinyDB for lightweight data storage. Overall, the system demonstrates core principles of secure financial systems, including confidentiality, integrity, authentication, and auditability.
